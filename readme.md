# Basic Calculator

#### A simple web-based calculator to perform basic arithmetic operations: addition, subtraction, multiplication, and division. This project is built with HTML, CSS, and JavaScript to demonstrate fundamental web development skills.

## Project Overview

### This calculator allows users to:

- Input numbers
- Perform basic calculations: addition, subtraction, multiplication, and division
- View results interactively on a simple, responsive interface

### This project showcases:

- HTML for structuring the calculator layout and buttons.
- CSS for styling the calculator to make it visually appealing and user-friendly.
- JavaScript for implementing the core functionality of the calculator, including handling button clicks and performing calculations.

### Usage

- Input Numbers: Click the number buttons to input your numbers.
- Select Operation: Choose an arithmetic operation (addition, subtraction, multiplication, or division).
- Get Result: Click the = button to display the calculation result.
- Clear: Use the C button to reset the display and start a new calculation.
